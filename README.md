# 仿微博
准备仿写微博作为自己的一个项目

写好了处理一些东西再公开吧


## 打算实现的功能
1. 查看用户信息
2. 显示关注用户动态
3. 显示热门动态
4. 显示与我相关的消息
5. 调用客户端或web页面发微博
